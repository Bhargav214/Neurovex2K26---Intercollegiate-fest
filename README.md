# Neurovex 2K26 - Intercollegiate Fest

A modern Next.js web application for the Neurovex 2K26 intercollegiate festival.

**Live Demo:** [https://neurovex2k26.vercel.app](https://neurovex2k26.vercel.app)

## 📋 Overview

Neurovex 2K26 is a web-based platform designed to manage and showcase an intercollegiate festival. Built with cutting-edge technologies, it provides a seamless user experience for event registration, management, and information dissemination.

## 🚀 Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) v16.1.6
- **UI Library:** React v18
- **Authentication:** JWT (jose v5.9.6)
- **API Integration:** Google APIs (googleapis v144.0.0)
- **Package Manager:** npm
